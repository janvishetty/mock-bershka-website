# Mock Bershka Website

This project is a responsive front-end clone inspired by the Bershka e-commerce website. It focuses on modern UI design, interactivity, and user experience using core web technologies.

## Overview

The application replicates key features of a fashion e-commerce platform, including product display, search functionality, wishlist handling, and quick view interactions. It is designed to demonstrate frontend development skills and responsive design principles.

## Features

- Product listing with structured layout
- Real-time search functionality
- Wishlist (favorites) using local storage
- Quick view modal for product details
- Dark/light theme toggle
- Fully responsive design

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

## Project Structure

- index.html – main structure of the application
- style.css – styling and layout
- script.js – interactivity and logic
- images/ – product and UI assets

## How to Run

1. Clone the repository:
   git clone https://github.com/janvishetty/mock-bershka-website.git

2. Open the project folder

3. Run the project:
   Open index.html in your browser

## Deployment

The project can be hosted using GitHub Pages for live access.
